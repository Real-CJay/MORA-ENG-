Basic Concepts

A differential equation contains an unknown function and its derivatives. The order is the highest derivative present. The degree is the power of that highest derivative.

Definition — ODE vs PDE
An ODE involves a function of one independent variable: $y'=\cos x$. A PDE involves two or more independent variables: $\partial u/\partial x+\partial u/\partial y=0$.
Definition — Order and Degree
Order: the highest derivative present. Degree: the power of that highest derivative, once the equation is polynomial in derivatives. Example: $(y'')^3+y=0$ has order 2, degree 3.
Classification by Linearity
An ODE is linear if it has the form $a_n(x)y^{(n)}+\cdots+a_1(x)y'+a_0(x)y=f(x)$. Key: the unknown $y$ and its derivatives appear linearly (no $y^2$, no $y\cdot y'$, no $\sin y$). Otherwise it is nonlinear.
Order & degree — easy mistakes

Order and degree can be confused. For $(y'')^3+2y'=x$: order is 2 (the $y''$ term), degree is 3 (the power on $y''$).

Degree is only defined when the ODE is polynomial in its derivatives. $\sin(y')=x$ has order 1 but no defined degree (transcendental in $y'$).

Solutions & Initial Value Problems

A solution is a function that satisfies the ODE identically. The general solution contains arbitrary constants. An initial value problem (IVP) pins down those constants.

Definition — General Solution
A function $y=\phi(x)$ is a solution of $y'=f(x,y)$ if $\phi'(x)=f(x,\phi(x))$ for all $x$ in the domain. The general solution contains as many arbitrary constants as the order of the ODE.
Definition — Initial Value Problem
ODE plus initial condition: $y'=f(x,y)$, $\;y(x_0)=y_0$. Substituting the initial condition into the general solution fixes the arbitrary constants to give the particular solution.

Separable First-Order Equations

An equation is separable if the right-hand side factors into an $x$-only part and a $y$-only part. Then the variables can be placed on opposite sides before integrating.

Definition — Separable Equation
$$\frac{dy}{dx}=f(x)\,g(y) \quad\Longrightarrow\quad \frac{dy}{g(y)}=f(x)\,dx \quad\Longrightarrow\quad \int\frac{dy}{g(y)}=\int f(x)\,dx$$
Method — Four Steps
  1. Move all $y$-terms (with $dy$) to the left.
  2. Move all $x$-terms (with $dx$) to the right.
  3. Integrate both sides; include one constant $C$.
  4. Apply initial condition if given to find $C$.
1. Separate: $\frac{1}{y}dy=x\,dx$.

2. Integrate: $\ln|y|=\frac{x^2}{2}+C_0$.

3. Solve for $y$: $y=Ce^{x^2/2}$ (absorbing $e^{C_0}$ into $C$).

Answer: $y=Ce^{x^2/2}$.
Warning
Only separate when the RHS truly factors into an $x$-only part times a $y$-only part. $dy/dx = x+y$ is NOT separable.

Homogeneous First-Order Equations

A first-order ODE is homogeneous (of degree 0) if the right-hand side is expressible purely as a function of $y/x$. The substitution $y=vx$ converts it to a separable equation.

Definition — Homogeneous Equation (degree 0)
$\dfrac{dy}{dx}=f(x,y)$ is homogeneous if $f(\lambda x,\lambda y)=f(x,y)$ (degree 0 in both together), equivalently $f(x,y)=F(y/x)$ for some function $F$.
Method — Substitution $y=vx$
Set $y=vx$, so $\dfrac{dy}{dx}=v+x\dfrac{dv}{dx}$. The ODE becomes: $$v+x\frac{dv}{dx}=F(v) \quad\Longrightarrow\quad x\frac{dv}{dx}=F(v)-v$$ This is separable in $v$ and $x$. Separate, integrate, then substitute back $v=y/x$.
Identify: $f=(x+y)/x=1+y/x=F(y/x)$ — homogeneous.

Substitute $y=vx$: $v+xv'=1+v$, so $xv'=1$, giving $v'=1/x$.

Integrate: $v=\ln|x|+C$.

Back-substitute: $y/x=\ln|x|+C$, so $\boxed{y=x(\ln|x|+C)}$.
Intuition
A homogeneous function of degree 0 is scale-invariant: doubling $x$ and $y$ gives the same value. So the ratio $y/x$ — not $x$ or $y$ individually — determines the RHS. The substitution replaces the non-separable pair $(x,y)$ with the separable pair $(x,v)$.

First-Order Linear Equations

The standard form $y'+P(x)y=Q(x)$ is solved by multiplying through by the integrating factor $\mu(x)=e^{\int P\,dx}$, which makes the left side a perfect derivative.

Standard Form
$$\frac{dy}{dx}+P(x)\,y=Q(x), \qquad \mu(x)=e^{\int P(x)\,dx}$$ After multiplication: $\dfrac{d}{dx}[\mu y]=\mu Q$, so $\mu y=\int\mu Q\,dx+C$.
Method — Integrating Factor
  1. Write the equation in standard form $y'+P(x)y=Q(x)$.
  2. Compute $\mu(x)=e^{\int P(x)\,dx}$.
  3. Multiply both sides by $\mu$.
  4. The LHS becomes $(\mu y)'$. Integrate both sides.
  5. Divide by $\mu$ to get $y$; apply initial condition.
We want to choose $\mu$ so that $\mu y'+\mu Py=(\mu y)'=\mu y'+\mu'y$. This requires $\mu'=\mu P$, i.e. $d\mu/dx=\mu P(x)$. Separating: $d\mu/\mu=P\,dx$, so $\ln\mu=\int P\,dx$, giving $\mu=e^{\int P\,dx}$.

Bernoulli Equation

The Bernoulli equation looks like a first-order linear equation but has a power of $y$ on the right. The substitution $v=y^{1-n}$ linearises it.

Bernoulli Form
$$\frac{dy}{dx}+P(x)\,y=f(x)\,y^n \qquad (n\ne 0,1)$$ Substitution $v=y^{1-n}$ turns this into a first-order linear equation in $v$.
Method
  1. Identify $P(x)$, $f(x)$, and $n$.
  2. Set $v=y^{1-n}$. Compute $v'=(1-n)y^{-n}y'$.
  3. Divide the original equation by $y^n$, then substitute $v$ and $v'$.
  4. Solve the resulting linear equation in $v$.
  5. Back-substitute $y=v^{1/(1-n)}$.
$n=2$, set $v=y^{-1}$. Then $v'=-y^{-2}y'$.

Divide the ODE by $y^2$: $y'/y^2+1/y=x$, i.e. $-v'+v=x$, giving $v'-v=-x$.

Integrating factor: $\mu=e^{-x}$. Then $(ve^{-x})'=-xe^{-x}$.

Integrate: $ve^{-x}=(x+1)e^{-x}+C$, so $v=x+1+Ce^x$.

Answer: $y=\dfrac{1}{x+1+Ce^x}$.
StepAction
1Identify Bernoulli form; read off $P$, $f$, $n$
2Set $v=y^{1-n}$
3Divide by $y^n$ and substitute to get a linear equation in $v$
4Solve with integrating factor; back-substitute

Second-Order Linear Equations

The standard homogeneous form $y''+ay'+by=0$ is solved by trying $y=e^{mx}$, which gives the auxiliary (characteristic) equation $m^2+am+b=0$. Three root cases give three solution types.

Auxiliary Equation
Try $y=e^{mx}$ in $y''+ay'+by=0$. This gives $m^2+am+b=0$.
Roots of auxiliary equationComplementary solution $y_c$
Two distinct real roots $m_1\ne m_2$$y_c=Ae^{m_1x}+Be^{m_2x}$
Repeated root $m$ (i.e. $m_1=m_2$)$y_c=(Ax+B)e^{mx}$
Complex roots $p\pm iq$ ($q>0$)$y_c=e^{px}(C\cos qx+D\sin qx)$
Physical Intuition
These correspond to a damped oscillator $y''+2\zeta\omega_0 y'+\omega_0^2 y=0$: distinct real roots → overdamped (exponential decay); repeated root → critically damped (fastest return to zero without oscillation); complex roots → underdamped (decaying oscillation).

Undetermined Coefficients

For $y''+ay'+by=f(x)$, the general solution is $y=y_c+y_p$, where $y_p$ is a particular solution guessed from the shape of $f(x)$ and its coefficients are fixed by substitution.

Guess Table
If $f(x)$ is...Guess $y_p$
Constant $k$Constant $A$
Polynomial $kx^n+\cdots$General polynomial of same degree
$k\sin\omega x$ or $k\cos\omega x$$A\sin\omega x+B\cos\omega x$
$ke^{\alpha x}$$Ae^{\alpha x}$, unless it overlaps with $y_c$
$ke^{\alpha x}\cos\beta x$ (or $\sin$)$e^{\alpha x}(A\cos\beta x+B\sin\beta x)$
Overlap rule
If your guess already appears in $y_c$, multiply the entire guess by $x$. Repeat if still overlapping. This ensures $y_p\notin\ker(L)$.
$y_c=C_1e^x+C_2e^{2x}$ (from previous example).

Guess $y_p=Ae^{3x}$ (no overlap: $3\ne 1,2$).

Substitute: $9Ae^{3x}-9Ae^{3x}+2Ae^{3x}=e^{3x}$, so $2A=1$, giving $A=\frac12$.

Answer: $y=C_1e^x+C_2e^{2x}+\frac12 e^{3x}$.

Wronskian & Linear Independence

Two solutions are linearly independent — and form a valid fundamental set — iff their Wronskian is not identically zero.

Definition — Wronskian
For two functions $u(x),v(x)$: $$W(u,v)=\begin{vmatrix}u&v\\u'&v'\end{vmatrix}=uv'-vu'$$ $W\not\equiv 0$ on the interval $\iff$ $u$ and $v$ are linearly independent.
Abel's Formula
For $y''+P(x)y'+Q(x)y=0$: $$W(x)=C\,e^{-\int P(x)\,dx}$$ So $W$ either never vanishes (if $C\ne 0$) or is identically zero — there is no middle ground.
If $u,v$ are solutions, differentiate $W=uv'-vu'$: $W'=uv''-vu''$. Using the ODE, $u''=-P u'-Qu$ and $v''=-Pv'-Qv$: $W'=u(-Pv'-Qv)-v(-Pu'-Qu)=-P(uv'-vu')=-PW.$ So $W'+PW=0$, giving $W=Ce^{-\int P\,dx}$.
Abel shortcut in practice

For $y''+P(x)y'+Q(x)y=0$, if you know one solution $y_1$, you can find $y_2$ using Abel's formula:

$$W=y_1 y_2'-y_2 y_1' = Ce^{-\int P\,dx}$$

This is a first-order linear ODE in $y_2$, solvable by the integrating factor method. The technique is called reduction of order.

Example: $y_1=e^x$ for $y''-y=0$

We know $y_1=e^x$ solves $y''-y=0$ ($P=0$). Abel gives $W=Ce^{-\int 0\,dx}=C$.

$W=y_1y_2'-y_2y_1'=e^x y_2'-e^x y_2=e^x(y_2'-y_2)=C$.

So $y_2'-y_2=Ce^{-x}$. Integrating factor: $\mu=e^{-x}$. Then $(y_2 e^{-x})'=Ce^{-2x}$, giving $y_2 e^{-x}=-\frac{C}{2}e^{-2x}+K$, so $y_2=-\frac{C}{2}e^{-x}+Ke^x$. Choosing constants: $y_2=e^{-x}$.

Variation of Parameters

This method finds a particular solution using two known independent homogeneous solutions $y_1,y_2$. It works for any $f(x)$, not just the special forms covered by undetermined coefficients.

Theorem — Variation of Parameters Formula
For $y''+p(x)y'+q(x)y=f(x)$, if $y_1,y_2$ solve the homogeneous equation and $W=y_1y_2'-y_2y_1'$: $$y_p=-y_1\int\frac{y_2\,f}{W}\,dx+y_2\int\frac{y_1\,f}{W}\,dx$$
Intuition
Instead of constants $A,B$ in $y=Ay_1+By_2$, let them vary: $y=u_1(x)y_1+u_2(x)y_2$. Imposing $u_1'y_1+u_2'y_2=0$ eliminates the $y'$ terms in the ODE. The remaining condition gives a system for $u_1',u_2'$ solvable by Cramer's rule using $W$.
Homogeneous solutions: $y_1=\cos x$, $y_2=\sin x$.

Wronskian: $W=\cos^2 x+\sin^2 x=1$.

Apply formula with $f=\tan x$:
$y_p=-\cos x\int\sin x\tan x\,dx+\sin x\int\cos x\tan x\,dx$
$=-\cos x\int\frac{\sin^2 x}{\cos x}\,dx+\sin x\int\sin x\,dx$

The second integral gives $-\sin x\cos x$. For the first, $\frac{\sin^2 x}{\cos x}=\frac{1-\cos^2 x}{\cos x}=\sec x-\cos x$, so $\int=\ln|\sec x+\tan x|-\sin x$.

Final: $y_p=-\cos x\,\ln|\sec x+\tan x|$ (the $\sin x\cos x$ terms cancel).
