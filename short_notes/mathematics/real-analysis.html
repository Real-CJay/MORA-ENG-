Order Matters: ∃x∀y vs ∀y∃x

Key Insight Think: x = doctor, y = village, P(x,y) = "doctor x can treat village y".

∃x∀y P(x,y) — one superhero doctor treats every village. That single doctor also works for each village individually, so the right side follows.

∀y∃x P(x,y) — every village has some doctor, but each village might have a different one. No single superhero doctor need exist.
Theorem
∃x∀y P(x,y) ⟹ ∀y∃x P(x,y) but the reverse does NOT hold in general.
Counterexample
Let P(x,y) = "x = y" over ℤ. Then ∀y∃x (x=y) is true — just pick x=y. But ∃x∀y (x=y) is false — no single integer equals every integer.

↳ This asymmetry reappears later in pointwise vs uniform convergence.

Intuition A set is a collection with no repetitions. Set operations are AND, OR, NOT in disguise.

Relations & Functions

Intuition A relation is any subset of A×B — a set of ordered pairs. A function is a relation that is everywhere-defined and not one-many.

Equivalence Relations

Intuition An equivalence relation is a custom equality — "these things are the same for our purposes." Think: same birthday, same remainder mod 3.
Definition 7 — Equivalence Relation
R: A→A is an equivalence relation iff:
1. Reflexive: ∀a, (a,a) ∈ R — you share your birthday with yourself 2. Symmetric: (a,b)∈R ⟹ (b,a)∈R — if Ali shares your birthday, you share Ali's 3. Transitive: (a,b)∈R ∧ (b,c)∈R ⟹ (a,c)∈R

Equivalence Classes = Partitions

Intuition An equivalence relation automatically sorts A into disjoint groups — partitions. Everyone in a group is equivalent to everyone else. Each element belongs to exactly one group.
Example — congruence mod 3 on ℤ
a ~ b iff (a − b) is divisible by 3 [0] = {…, −6, −3, 0, 3, 6, …} remainder 0 [1] = {…, −5, −2, 1, 4, 7, …} remainder 1 [2] = {…, −4, −1, 2, 5, 8, …} remainder 2 All of ℤ partitioned into 3 disjoint buckets. Note: [0] = [3] = [6] = [−3] — same bucket, different names.

Equivalence relation ↔ partition are two languages for the same thing. Give me a partition → I write an equivalence relation, and vice versa.

(S∘R)⁻¹ = R⁻¹∘S⁻¹

Intuition Socks-and-shoes: put on socks then shoes. To reverse: shoes off first, then socks. Order flips when you invert.
Let R: A→B, S: B→C. First note ran(S⁻¹) = B = preran(R⁻¹), so R⁻¹∘S⁻¹ exists.
Left ⊆ Right

Numbers & Completeness

Intuition ℝ is defined axiomatically. The three axiom groups — Field, Order, Completeness — together characterise the real line uniquely.

Field Axioms (Axiom 1)

Key field axioms
+ is commutative, associative; additive identity 0; inverses −a · is commutative, associative; multiplicative identity 1; inverses a⁻¹ for a≠0 · distributes over +

Order Axioms (Axiom 2)

Order axioms
Trichotomy: exactly one of a=b, a<b, b<a holds Transitivity: a<b ∧ b<c ⟹ a<c + compatible: a<b ⟹ a+c<b+c · compatible: a<b ∧ 0<c ⟹ ac<bc

Completeness Axiom (Axiom 3)

Intuition ℚ has gaps — √2 is missing. ℝ fills every gap. Completeness is the axiom that defines what makes ℝ different from ℚ. It says: wherever your data tries to arrive, that destination exists in ℝ.
Axiom 3 — Completeness
Every non-empty subset of ℝ bounded above has a supremum in ℝ. Every non-empty subset bounded below has an infimum in ℝ.

Supremum & Infimum

Definition 15
sup A = least upper bound = min{set of upper bounds} inf A = greatest lower bound = max{set of lower bounds} Key: u = sup A ⟺ ∀ε>0, ∃a∈A: a+ε > u (u is reachable from below within any ε)

Three Equivalent Forms of Completeness

Intuition All three say the same thing — ℝ has no holes — from three angles: sets, clustering sequences, and bounded sequences.
(1) Sup version: every bounded subset has sup/inf in ℝ ⟺ (2) Cauchy version: every Cauchy sequence in ℝ converges in ℝ ⟺ (3) Bolzano-Weierstrass: every bounded sequence has a convergent subsequence

In ℚ all three fail at the same holes (√2, π, e, …). ℝ has none of these holes.

Cauchy Sequences

Intuition A Cauchy sequence is one where terms get close to each other — not to a known limit. In ℝ, if terms cluster together, they must be heading somewhere real.
Definition 23
uₙ is Cauchy iff ∀ε>0, ∃N, ∀m,n>N: |uₙ−uₘ| < ε In ℝ: Cauchy ⟺ Convergent In ℚ: Cauchy ⟹ might not converge (e.g. 1.4, 1.41, 1.414, … → √2 ∉ ℚ)

Bolzano-Weierstrass Theorem

Intuition Infinite sequence, finite box. Infinitely many things crammed into finite space — they cannot all stay far apart. Something must pile up. Completeness guarantees the pile-up point exists in ℝ.
Theorem 8 — Bolzano-Weierstrass
Every bounded sequence has a convergent subsequence.
Formal proof sketch
  • Trap infinitely many terms inside smaller and smaller closed intervals.
  • Choose one sequence term from each nested interval with increasing indices.
  • The interval lengths go to 0, so the chosen subsequence is forced to converge.
Strict formal proof
1.Since (uₙ) is bounded, choose α,β with α ≤ uₙ ≤ β for all n; set I₀=[α,β].boundedness
2.Bisect I₀. At least one half contains uₙ for infinitely many n. Call that half I₁. Repeat inductively.pigeonhole principle
3.Obtain closed nested intervals I₀ ⊇ I₁ ⊇ I₂ ⊇ ··· with length |Iₖ|=(β−α)/2ᵏ and infinitely many terms in each Iₖ.construction
4.Choose n₁<n₂<··· so that u_{nₖ} ∈ Iₖ for every k.possible because Iₖ has infinitely many later terms
5.By nested interval completeness, ⋂ₖ Iₖ = {L} for some L∈ℝ.completeness of ℝ
6.Given ε>0, choose K with |I_K|<ε. If k≥K, then u_{nₖ}∈Iₖ⊆I_K and L∈I_K, hence |u_{nₖ}−L|≤|I_K|<ε.ε proof of convergence
7.Therefore u_{nₖ} → L, so (uₙ) has a convergent subsequence.definition of subsequential convergence
∎

Monotone Sequences & Bounded Convergence

Intuition A monotone increasing sequence is like climbing stairs — if there's a ceiling, you must stop somewhere. Completeness guarantees that "somewhere" is a real number.
Definition 22 — Monotone Sequences
Increasing: ∀m,n, n>m ⟹ uₙ ≥ uₘ (strict: uₙ > uₘ) Decreasing: ∀m,n, n>m ⟹ uₙ ≤ uₘ (strict: uₙ < uₘ) Monotone = either increasing or decreasing.
Theorem 3 — Bounded Monotone Convergence
1. Increasing + bounded above ⟹ converges to sup{uₙ} (from below) 2. Decreasing + bounded below ⟹ converges to inf{uₙ} (from above)
Formal proof sketch
  • For an increasing bounded sequence, the supremum is the only possible limit.
  • The supremum property puts some term above s−ε.
  • Monotonicity keeps all later terms between s−ε and s.
Strict formal proof
1.Assume (uₙ) is increasing and bounded above. Let S={uₙ:n∈ℕ}. Since S is non-empty and bounded above, s=sup S exists.Completeness Axiom
2.For every ε>0, s−ε is not an upper bound of S.definition of supremum
3.Hence ∃N such that u_N > s−ε.not an upper bound
4.For n≥N, monotonicity gives uₙ ≥ u_N > s−ε, and since s is an upper bound, uₙ ≤ s.increasing + upper bound
5.Thus 0 ≤ s−uₙ < ε, i.e. |uₙ−s|<ε for all n≥N.ε–N convergence
6.Therefore uₙ→s=sup{uₙ}. The decreasing bounded-below case follows by applying the increasing case to −uₙ, giving limit inf{uₙ}.symmetry
∎
Theorem 6 — Subsequences Inherit the Limit
If uₙ → L, then every subsequence u_{mₙ} → L. Converse also true: if every subsequence → L, then uₙ → L.
Formal proof sketch
  • A subsequence only skips terms; after the whole sequence enters an ε-neighbourhood, every later chosen term is also inside.
  • For the converse, if the whole sequence fails to converge to L, one can build a bad subsequence staying outside some ε₀-neighbourhood of L.
Strict formal proof
1.Assume uₙ→L. Let (u_{mₖ}) be any subsequence, where mₖ is strictly increasing.setup
2.Given ε>0, choose N such that n≥N ⇒ |uₙ−L|<ε.convergence of uₙ
3.Because mₖ≥k, choose K≥N. Then k≥K ⇒ mₖ≥N ⇒ |u_{mₖ}−L|<ε.subsequence indices go to infinity
4.Therefore every subsequence converges to L.definition
5.Conversely, suppose every subsequence converges to L but uₙ does not converge to L.contradiction setup
6.Then ∃ε₀>0 such that ∀N ∃n≥N with |uₙ−L|≥ε₀.negation of convergence
7.Choose n₁ with |u_{n₁}−L|≥ε₀, then choose n_{k+1}>nₖ with |u_{n_{k+1}}−L|≥ε₀.recursive construction
8.The subsequence (u_{nₖ}) stays outside the ε₀-neighbourhood of L, so it cannot converge to L, contradiction.violates assumption
9.Hence uₙ→L.conclusion
∎
Intuition If the whole sequence settles near L, any selection of terms also settles near L — you can't escape a limit by skipping terms.
Theorem 7 — Monotone Subsequence Theorem
Every sequence has a monotone subsequence.
Formal proof sketch
  • Call an index a peak if its term is at least every later term.
  • Infinitely many peaks give a decreasing subsequence.
  • Only finitely many peaks means after some point every term has a later larger term, so recursively choose an increasing subsequence.
Strict formal proof
1.Call n a peak if uₙ ≥ u_m for every m≥n.definition
2.If there are infinitely many peaks n₁<n₂<···, then for k<j we have u_{nₖ} ≥ u_{nⱼ}.peak property
3.Thus (u_{nₖ}) is decreasing, hence monotone.case 1
4.If there are only finitely many peaks, choose N after the last peak.case 2 setup
5.For every n≥N, n is not a peak, so ∃m>n with u_m>uₙ.negation of peak
6.Choose n₁≥N. Having chosen nₖ, choose n_{k+1}>nₖ with u_{n_{k+1}}>u_{nₖ}.recursion
7.Then (u_{nₖ}) is strictly increasing, hence monotone.case 2 conclusion
8.In either case the original sequence has a monotone subsequence.exhaustion of cases
∎
Theorem 9 — Cauchy + Convergent Subsequence
If uₙ is Cauchy and u_{mₙ} → L (some subsequence converges), then uₙ → L.
Formal proof sketch
  • Cauchy means all sufficiently late terms are close to one another.
  • The convergent subsequence supplies one late term close to L.
  • Triangle inequality then drags every late term close to L.
Strict formal proof
1.Let ε>0.arbitrary ε
2.Since (uₙ) is Cauchy, choose N₁ such that m,n≥N₁ ⇒ |uₙ−u_m|<ε/2.Cauchy condition
3.Since u_{mₖ}→L, choose K such that k≥K ⇒ |u_{mₖ}−L|<ε/2.subsequence convergence
4.Choose k≥K with mₖ≥N₁.subsequence indices tend to infinity
5.For n≥N₁, |uₙ−L| ≤ |uₙ−u_{mₖ}| + |u_{mₖ}−L| < ε/2+ε/2=ε.triangle inequality
6.Therefore uₙ→L.definition of convergence
∎
Intuition The Cauchy condition makes all terms cluster together. If you know one cluster point L exists (from the subsequence), all other terms must be near it too — they're all near each other by Cauchy.

Three Equivalent Completeness Conditions

Intuition Completeness is a single idea expressed three different ways. They're all saying the same thing: ℝ has no holes. ℚ fails all three at the same holes (√2, etc).
Theorem 10 — TFAE for any A ⊂ ℝ
The following are equivalent: 1. Completeness Axiom: every non-empty bounded subset has sup and inf in A 2. Bolzano-Weierstrass: every bounded sequence in A has a convergent subsequence in A 3. Cauchy Completeness: every Cauchy sequence in A converges to a point in A
Formal proof sketch
  • The completeness axiom gives monotone convergence, which gives Bolzano-Weierstrass.
  • Bolzano-Weierstrass turns every bounded Cauchy sequence into a convergent one.
  • Cauchy completeness recovers the supremum by bisection.
Strict formal proof
1.Assume every non-empty bounded subset of A has its supremum/infimum in A.(1)
2.A bounded sequence in A has a monotone subsequence by the Monotone Subsequence Theorem.Theorem 7
3.That monotone subsequence is bounded, hence converges to its sup or inf, which lies in A by (1). Thus Bolzano-Weierstrass holds in A.(1) ⇒ (2)
4.Assume Bolzano-Weierstrass in A and let (uₙ) be Cauchy in A.(2)
5.Every Cauchy sequence is bounded; hence (uₙ) has a convergent subsequence u_{nₖ}→L∈A.BW
6.By Theorem 9, a Cauchy sequence with a convergent subsequence converges to the same L. Thus Cauchy completeness holds.(2) ⇒ (3)
7.Assume Cauchy completeness in A. Let B⊂A be non-empty and bounded above in A. Choose v₀∈B and an upper bound u₀∈A.(3) setup
8.Define wₙ=(uₙ+vₙ)/2. If wₙ is an upper bound of B, set u_{n+1}=wₙ, v_{n+1}=vₙ; otherwise set u_{n+1}=uₙ and choose v_{n+1}∈B with v_{n+1}>wₙ.bisection
9.Then uₙ is always an upper bound of B, vₙ∈B, and 0≤uₙ−vₙ≤(u₀−v₀)/2ⁿ.induction
10.Hence (uₙ) and (vₙ) are Cauchy and converge in A to the same s.Cauchy completeness
11.Because each uₙ is an upper bound and uₙ→s, s is an upper bound of B. Because vₙ∈B and vₙ→s, no number <s can be an upper bound. Thus s=sup B.supremum verification
12.The infimum statement follows by applying the same argument to −B.symmetry
∎
For a general subset A, read this as the completeness property relative to A; for the main course setting A=ℝ, this is the usual completeness theorem.

Convergence Tests

Limit Comparison (Thm 12)
lim uₙ/vₙ = R ∈ (0,∞): same fate (both conv or div) R = 0: vₙ conv ⟹ uₙ conv R = ∞: vₙ div ⟹ uₙ div
Formal proof sketch
  • A positive finite limiting ratio means uₙ and vₙ eventually differ only by fixed constant factors.
  • Constant factors do not change convergence of positive series.
Strict formal proof
1.Assume uₙ,vₙ>0 and lim(uₙ/vₙ)=R with 0<R<∞.hypothesis
2.Choose N such that n≥N ⇒ R/2 < uₙ/vₙ < 3R/2.definition of limit
3.Thus (R/2)vₙ < uₙ < (3R/2)vₙ for all n≥N.multiply by vₙ>0
4.If Σvₙ converges, then Σuₙ converges by direct comparison with (3R/2)Σvₙ.comparison
5.If Σvₙ diverges, then Σuₙ diverges because uₙ ≥ (R/2)vₙ eventually.comparison
6.The R=0 and R=∞ one-sided versions follow from the same inequalities after choosing uₙ/vₙ<C or uₙ/vₙ>C eventually.variants
∎
Alternating Series (Thm 18)
Σ(-1)ⁿ⁻¹uₙ, uₙ > 0: if uₙ decreasing and uₙ → 0 then the series converges.
Formal proof sketch
  • Even and odd partial sums squeeze the same number.
  • The gap between the two squeezing sequences is the next term, which tends to 0.
Strict formal proof
1.Let Sₙ=Σ_{k=1}^n (-1)^{k-1}uₖ with uₖ≥0, uₖ decreasing, and uₖ→0.setup
2.S_{2m}= (u₁−u₂)+···+(u_{2m-1}−u_{2m}) is increasing in m and bounded above by S_{2m+1}.monotone structure
3.S_{2m+1}=S_{2m}+u_{2m+1} is decreasing in m and bounded below by S_{2m}.monotone structure
4.By bounded monotone convergence, S_{2m}→A and S_{2m+1}→B.MCT
5.0≤S_{2m+1}−S_{2m}=u_{2m+1}→0, hence A=B.gap tends to 0
6.Since both even and odd subsequences of partial sums converge to the same limit, Sₙ converges.sequence convergence criterion
∎
Direct Comparison
0 < uₙ < vₙ: ∑vₙ conv ⟹ ∑uₙ conv ∑uₙ div ⟹ ∑vₙ div
Formal proof sketch
  • For positive terms, partial sums are increasing.
  • If the larger partial sums are bounded, the smaller partial sums are bounded.
  • If the smaller partial sums blow up, the larger ones must also blow up.
Strict formal proof
1.Assume 0≤uₙ≤vₙ for all sufficiently large n.eventual comparison
2.If Σvₙ converges, then its partial sums are bounded above.positive series
3.For N≤m, Σ_{n=N}^m uₙ ≤ Σ_{n=N}^m vₙ ≤ Σ_{n=N}^∞ vₙ.comparison of tails
4.Thus the tail partial sums of Σuₙ are increasing and bounded, hence converge.MCT
5.Therefore Σuₙ converges.finite initial terms do not matter
6.If Σuₙ diverges and Σvₙ converged, the first part would imply Σuₙ converges, contradiction. Hence Σvₙ diverges.contrapositive
∎
Ratio Test (Thm 15)
R = lim uₙ₊₁uₙ R < 1 → conv R > 1 → div R = 1 → fails
Formal proof sketch
  • Eventually the terms shrink at a geometric rate if the ratio limit is below 1.
  • If the ratio limit is above 1, the terms do not even go to 0.
  • At ratio 1, the test contains no information.
Strict formal proof
1.Let r=lim |u_{n+1}/uₙ|.setup
2.If r<1, choose q with r<q<1 and choose N such that n≥N ⇒ |u_{n+1}|≤q|uₙ|.definition of limit
3.Inductively, |u_{N+k}|≤q^k|u_N| for k≥0.geometric domination
4.Hence Σ_{k=0}^∞ |u_{N+k}| ≤ |u_N|Σq^k <∞, so Σuₙ converges absolutely.comparison with geometric series
5.If r>1, choose q>1 with eventually |u_{n+1}|≥q|uₙ|.definition of limit
6.Then |uₙ| cannot tend to 0; therefore Σuₙ diverges.term test
7.If r=1, examples Σ1/n and Σ1/n² both have ratio limit 1 but different fates; no conclusion follows.sharpness
∎
Root Test (Thm 16)
R = lim uₙ1n R < 1 → conv R > 1 → div Root is stronger than ratio.
Formal proof sketch
  • The nth root measures the hidden geometric rate.
  • Below 1 gives geometric domination.
  • Above 1 gives terms that fail to approach 0 along infinitely many indices.
Strict formal proof
1.Let r=limsup |uₙ|^{1/n}.root-test form
2.If r<1, choose q with r<q<1. Then ∃N such that n≥N ⇒ |uₙ|^{1/n}≤q.limsup property
3.Hence |uₙ|≤qⁿ for n≥N.raise to n
4.Since Σqⁿ converges, Σuₙ converges absolutely by comparison.geometric comparison
5.If r>1, then for infinitely many n, |uₙ|^{1/n}>1.limsup property

Intuition The alternating series test is the simplest case: signs oscillate, partial sums stay bounded. Dirichlet generalises to any sequence bₖ whose partial sums stay bounded — combined with a decreasing aₖ→0, the oscillation and the shrinking envelope force convergence.

  • Bounded partial sums of bₖ control the oscillatory part.
  • The decreasing factor aₖ→0 makes the remaining bound vanish.

2.For q>p, summation by parts gives Σ_{k=p+1}^q aₖbₖ = a_qB_q − a_{p+1}B_p + Σ_{k=p+1}^{q-1}(aₖ−a_{k+1})Bₖ.Abel summation

Intuition A power series ∑aₙ(x−c)ⁿ is a function of x. R is the radius of validity — inside R, with enough terms, the partial sum approximates the function exactly. Outside R, it diverges regardless of how many terms you add.

Continuity

Intuition Continuity is the limit condition with the box center sitting ON the graph. Three things must align: f(a) exists, the limit exists, and they agree.
Definition 32 — Continuity
f is continuous at a iff lim f(x) = f(a) x→a Equivalently: ∀ε>0, ∃δ>0, |x−a|<δ ⟹ |f(x)−f(a)|<ε
Theorem 27 — Sequential Continuity
f is continuous at a iff for every sequence xₙ→a: f(xₙ)→f(a). Use this to disprove continuity: find one bad sequence.
Formal proof sketch
  • Continuity sends every convergent input sequence to the corresponding output sequence.
  • If ε–δ continuity fails, choose xₙ within 1/n of a but with output staying at least ε₀ away.
Strict formal proof
1.Assume f is continuous at a and let xₙ→a.forward direction setup
2.Given ε>0, choose δ>0 such that |x−a|<δ ⇒ |f(x)−f(a)|<ε.continuity at a
3.Since xₙ→a, choose N such that n≥N ⇒ |xₙ−a|<δ.sequence convergence
4.Then n≥N ⇒ |f(xₙ)−f(a)|<ε, so f(xₙ)→f(a).definition
5.Conversely, assume every sequence xₙ→a satisfies f(xₙ)→f(a), but f is not continuous at a.contradiction setup
6.Then ∃ε₀>0 such that ∀δ>0 ∃x with |x−a|<δ and |f(x)−f(a)|≥ε₀.negation of continuity
7.For δ=1/n choose xₙ with |xₙ−a|<1/n and |f(xₙ)−f(a)|≥ε₀.choice for each n
8.Then xₙ→a, but f(xₙ) does not converge to f(a), contradiction.violates sequential assumption
9.Therefore f is continuous at a.conclusion
∎

Hierarchy: Lipschitz → Uniform → Continuous

Lipschitz: |f(x)−f(y)| ≤ L|x−y| for some fixed L ⟹ slope of every secant line bounded by L ⟸ a common sufficient condition: differentiable with bounded derivative Uniformly continuous: ∀ε>0, ∃δ>0, ∀x,y: |x−y|<δ ⟹ |f(x)−f(y)|<ε ⟹ one δ works for ALL x simultaneously ⟹ 1/x on (0,1) fails — needs smaller δ near 0 Continuous: δ can depend on x Each arrow is strict: √x on [0,1] is uniform but not Lipschitz.

The Three Big Theorems on [a,b]

Intuition [a,b] gives you three things: closed (endpoints exist), bounded (finite interval), connected (no gaps). Continuity links f's behaviour to all three.
Theorem 28 — Boundedness
f continuous on [a,b] ⟹ f is bounded: ∃M, |f(x)| ≤ M for all x.
Formal proof sketch
  • If a continuous function on [a,b] were unbounded, choose points where |f| grows beyond n.
  • Compactness gives a convergent subsequence of those points.
  • Continuity at the subsequential limit contradicts the outputs going to infinity.
Strict formal proof
1.Assume f is continuous on [a,b] and suppose f is not bounded.contradiction setup
2.For each n∈ℕ choose xₙ∈[a,b] with |f(xₙ)|≥n.unboundedness
3.The sequence (xₙ) is bounded, so by Bolzano-Weierstrass it has a subsequence x_{nₖ}→c∈[a,b].compact interval
4.By continuity, f(x_{nₖ})→f(c).sequential continuity
5.Every convergent real sequence is bounded, so (f(x_{nₖ})) is bounded.basic sequence theorem
6.But |f(x_{nₖ})|≥nₖ→∞, contradiction.choice of xₙ
7.Hence f is bounded on [a,b].conclusion
∎
Theorem 29 — Extreme Value
f continuous on [a,b] ⟹ f attains its maximum and minimum.
Formal proof sketch
  • Boundedness gives a finite supremum and infimum of the image.
  • Choose points whose function values approach the supremum.
  • Bolzano-Weierstrass plus continuity turns the supremum into an actually attained value.
Strict formal proof
1.By Theorem 28, f([a,b]) is bounded. Let M=sup f([a,b]).boundedness + completeness
2.For each n choose xₙ∈[a,b] such that f(xₙ)>M−1/n.definition of supremum
3.By Bolzano-Weierstrass, a subsequence x_{nₖ} converges to some c∈[a,b].compactness of [a,b]
4.By continuity, f(x_{nₖ})→f(c).sequential continuity
5.Since f(x_{nₖ})>M−1/nₖ and f(x_{nₖ})≤M, squeeze gives f(x_{nₖ})→M.squeeze
6.Therefore f(c)=M, so f attains its maximum.uniqueness of limits
7.Apply the same argument to −f to obtain a point where f attains its minimum.minimum
∎
Theorem 30 — Intermediate Value (IVT)
f continuous on [a,b], u strictly between f(a) and f(b) ⟹ ∃c∈(a,b): f(c)=u.

In one line: completeness (no holes in ℝ) + continuity (no jumps in f) = no holes in the image.
Formal proof sketch
  • Let S be the set of points where f is still below the target level u.
  • Completeness gives c=sup S.
  • Continuity rules out f(c)<u and f(c)>u, so f(c)=u.
Strict formal proof
1.Assume f(a)<u<f(b). Define S={x∈[a,b] : f(x)<u}.without loss of generality
2.S is non-empty because a∈S, and S is bounded above by b. Let c=sup S.completeness
3.We show f(c)=u.goal
4.If f(c)<u, continuity gives δ>0 such that |x−c|<δ ⇒ f(x)<u.continuity with margin u−f(c)
5.Then some x>c lies in S, contradicting c=sup S.not an upper bound
6.If f(c)>u, continuity gives δ>0 such that |x−c|<δ ⇒ f(x)>u.continuity with margin f(c)−u
7.Then no point of S lies in (c−δ,c], contradicting that c is the least upper bound of S.supremum must be approached from below
8.Thus neither f(c)<u nor f(c)>u, hence f(c)=u.trichotomy
9.The case f(b)<u<f(a) follows by applying the above to −f.symmetry
∎

Differentiation

Intuition The derivative is the slope of the tangent line — rise over run as the run shrinks to zero. Differentiability is a stronger condition than continuity: it requires the slope to exist (no corners).
Definition 34 — Derivative
f'(a) = lim f(x)−f(a)x−a = lim f(a+h)−f(a)h x→a h→0 f is differentiable at a iff this limit exists and is finite. Differentiable ⟹ Continuous (but NOT vice versa: |x| at 0)

Mean Value Theorem

Intuition If you drive 60 miles in 1 hour, at some moment your speedometer read exactly 60 mph. The instantaneous slope somewhere equals the average slope across the whole interval.
Theorem 35 — MVT
f continuous on [a,b], differentiable on (a,b) ⟹ ∃c∈(a,b): f'(c) = f(b)−f(a)b−a
Formal proof sketch
  • Subtract the secant line from f so the new function has equal endpoint values.
  • Rolle’s theorem then forces a zero derivative somewhere.
  • That zero derivative is exactly the MVT slope equation.
Strict formal proof
1.Define g(x)=f(x)−f(a)−[(f(b)−f(a))/(b−a)](x−a).subtract secant line
2.Then g is continuous on [a,b] and differentiable on (a,b).inherited from f
3.Also g(a)=0 and g(b)=0.secant line matches endpoints
4.By Rolle’s theorem, ∃c∈(a,b) such that g′(c)=0.Rolle
5.But g′(c)=f′(c)−[f(b)−f(a)]/(b−a).differentiate
6.Hence f′(c)=[f(b)−f(a)]/(b−a).rearrange
∎

Theorem 36 — Cauchy MVT

Formal proof sketch
  • Build a linear combination of f and g whose endpoint values agree.
  • Apply Rolle’s theorem to that combination.
  • Rearrange the zero derivative equation.

7.Since g′(c)≠0 and g(b)−g(a)≠0, f′(c)/g′(c)=[f(b)−f(a)]/[g(b)−g(a)].rearrange

Formal proof sketch
  • For each nearby x, Cauchy MVT converts f(x)/g(x) into f′(cₓ)/g′(cₓ).
  • As x approaches a, the Cauchy point cₓ also approaches a.
  • Therefore the derivative ratio limit transfers to the original ratio.

2.For x>a close to a, apply Cauchy MVT to f and g on [a,x].CMVT
3.There exists cₓ∈(a,x) such that [f(x)−f(a)]/[g(x)−g(a)] = f′(cₓ)/g′(cₓ).CMVT conclusion

Theorem 38 — 2nd Order MVT

Intuition If you know a function's value and all its derivatives at one point a, you can predict its value at all nearby points. Each term in the Taylor polynomial is engineered to match exactly one derivative at a, and nothing else.

Building the Polynomial

T(x) = f(a) + f'(a)(x−a) + f''(a)2!(x−a)2 + f'''(a)3!(x−a)3 + … = ∞ Σ f⁽ᵏ⁾(a)k! · (x−a)k k=0 Each term is engineered to match exactly one derivative of f at a.

Why Divide by k!?

Key Insight Differentiating (x−a)ᵏ exactly k times produces k! as a byproduct — the exponents cascade down: k·(k−1)·…·1. Dividing by k! cancels that cascade exactly, so the kth term contributes precisely f⁽ᵏ⁾(a) to the kth derivative and zero to all others.
dk/dxk [(x−a)k] = k! ← cascade from k differentiations coefficient = f⁽ᵏ⁾(a)k! ← cancels the cascade ⟹ k-th derivative of T at a = f⁽ᵏ⁾(a) ✓

Intuition The derivative form has a mysterious ζ — you know it exists but can't find it. The integral form replaces ζ with an explicit integral. No hidden ζ — it accumulates f⁽ⁿ⁺¹⁾ across the whole interval, weighted by distance from x.

Generalised MVT for Integrals

Intuition The weighted average of f over [a,b] (using weight g) must equal f at some interior point ζ. Proof: EVT gives min m and max M; the weighted average sits between them; IVT finds the ζ where f equals that average.
Generalised MVT for Integrals
f, g ∈ C[a,b], g does not change sign. Then ∃ζ ∈ [a,b] s.t. ∫ₐᵇ f(x)g(x) dx = f(ζ) · ∫ₐᵇ g(x) dx Recovering derivative form: set g(t) = (x−t)ⁿ ≥ 0: ∫ₐˣ f⁽ⁿ⁺¹⁾(t)(x−t)ⁿ dt = f⁽ⁿ⁺¹⁾(ζ)·∫ₐˣ(x−t)ⁿ dt = f⁽ⁿ⁺¹⁾(ζ)·(x−a)ⁿ⁺¹n+1
Formal proof sketch
  • A weighted average of f must lie between the minimum and maximum of f.
  • Continuity guarantees f actually takes every value between its minimum and maximum.
  • That value is f(ζ).
Strict formal proof
1.Assume g≥0 on [a,b]. If ∫_a^b g=0, then g≡0 by continuity and the identity is trivial.zero-weight case
2.Let m=min_{[a,b]} f and M=max_{[a,b]} f.EVT
3.Since m≤f(x)≤M and g(x)≥0, we have mg(x)≤f(x)g(x)≤Mg(x).multiply by nonnegative g
4.Integrate: mG≤I≤MG, where G=∫_a^b g(x)dx>0 and I=∫_a^b f(x)g(x)dx.monotonicity of integral
5.Divide by G: m≤I/G≤M.G positive
6.By IVT, since f is continuous and attains m and M, there exists ζ∈[a,b] with f(ζ)=I/G.intermediate value
7.Therefore I=f(ζ)G, i.e. ∫fg=f(ζ)∫g.rearrange
8.If g≤0, apply the proved case to −g.sign change
∎

Uniform Convergence

Intuition Pointwise convergence: each x converges at its own pace with its own N. Uniform convergence: one N governs all x simultaneously — N depends only on ε, not on x. The entire graph of uₙ drops inside the ε-tube at once.
Definition 37
Pointwise: ∀ε>0, ∀x, ∃N(ε,x), ∀n>N: |uₙ(x)−u(x)| < ε N depends on ε and x
Uniform: ∀ε>0, ∃N(ε), ∀x, ∀n>N: |uₙ(x)−u(x)| < ε N depends only on ε
Uniform ⟹ Pointwise  (not vice versa).

Supremum Norm Characterisation

Theorem 42
||uₙ−u||∞ = sup_x |uₙ(x)−u(x)| (worst-case gap) uₙ → u uniformly iff ||uₙ−u||∞ → 0
Formal proof sketch
  • Uniform convergence is exactly convergence of the worst-case error.
  • The supremum norm records that worst-case error.
Strict formal proof
1.Let E be the common domain and define ||uₙ−u||∞=sup_{x∈E}|uₙ(x)−u(x)|.definition
2.Assume uₙ→u uniformly. Given ε>0, choose N such that n≥N ⇒ |uₙ(x)−u(x)|<ε for every x∈E.uniform convergence
3.Then n≥N ⇒ sup_{x∈E}|uₙ(x)−u(x)|≤ε, so ||uₙ−u||∞→0.supremum bound
4.Conversely, assume ||uₙ−u||∞→0. Given ε>0, choose N such that n≥N ⇒ ||uₙ−u||∞<ε.norm convergence
5.Then for every x∈E and n≥N, |uₙ(x)−u(x)|≤||uₙ−u||∞<ε.definition of supremum
6.Thus uₙ→u uniformly.definition
∎

Why Uniform Convergence Preserves Continuity

Intuition To prove u is continuous at a, bound |u(x)−u(a)| using the triangle inequality — three ε/3 pieces. Uniform convergence lets you fix n before choosing x, so all three pieces can be made small simultaneously.
Theorem 44 — Uniform Limit Preserves Continuity
If each uₙ is continuous on E and uₙ → u uniformly on E, then u is continuous on E.
Formal proof sketch
  • Add and subtract uₙ(x) and uₙ(a).
  • Uniform convergence controls the two outer terms with one N for all x.
  • Continuity of the fixed uₙ controls the middle term.
Strict formal proof
1.Fix a∈E and ε>0.goal: continuity at a
2.By uniform convergence, choose N such that for all x∈E, |u_N(x)−u(x)|<ε/3.outer terms
3.Because u_N is continuous at a, choose δ>0 such that |x−a|<δ ⇒ |u_N(x)−u_N(a)|<ε/3.middle term
4.For |x−a|<δ, |u(x)−u(a)| ≤ |u(x)−u_N(x)| + |u_N(x)−u_N(a)| + |u_N(a)−u(a)|.triangle inequality
5.Each term on the right is <ε/3, so |u(x)−u(a)|<ε.combine
6.Therefore u is continuous at a. Since a was arbitrary, u is continuous on E.conclusion
∎

Swapping Limits and Integrals

Theorem 45 — swap lim and ∫
uₙ → u uniformly on [a,b] ⟹ lim ∫uₙ dx = ∫u dx = ∫lim uₙ dx
Formal proof sketch
  • Uniform convergence turns the difference of integrals into the integral of a uniformly small error.
  • The interval length multiplies the worst-case error.
  • The worst-case error tends to 0.
Strict formal proof
1.Assume uₙ and u are Riemann integrable on [a,b] and uₙ→u uniformly.hypothesis
2.For each n, |∫_a^b uₙ − ∫_a^b u| = |∫_a^b (uₙ−u)|.linearity
3.By the integral inequality, |∫_a^b (uₙ−u)| ≤ ∫_a^b |uₙ−u|.absolute value inequality
4.Since |uₙ(x)−u(x)|≤||uₙ−u||∞ for all x, ∫_a^b |uₙ−u| ≤ (b−a)||uₙ−u||∞.sup bound
5.Uniform convergence is equivalent to ||uₙ−u||∞→0.Theorem 42
6.Therefore |∫uₙ−∫u|→0, i.e. lim∫uₙ=∫u.conclusion
∎

Swapping Limits and Derivatives

Intuition Derivatives are sensitive — uniform convergence of the functions alone is not enough. You need the derivatives themselves to converge uniformly. The single convergence point x₀ anchors the height (shape + one point = unique function).
Theorem 46 — swap lim and d/dx
Conditions: 1. uₙ(x₀) converges for some x₀ ∈ [a,b] 2. u'ₙ(x) converges uniformly on [a,b] Then: uₙ(x) → u(x) uniformly AND u'(x) = lim u'ₙ(x) Why condition 2 is stricter: f(x) and f(x)+5 have identical derivatives — shape ≠ function. Need height anchor (condition 1) + shape convergence (condition 2).
Formal proof sketch
  • Uniform convergence of derivatives gives a limiting derivative v.
  • The convergence at one point fixes the vertical constant.
  • Integrate the derivatives from x₀ to x and pass the limit through the integral.
Strict formal proof
1.Let v be the uniform limit of u′ₙ on [a,b], and suppose uₙ(x₀)→A.hypotheses
2.For any x∈[a,b], FTC gives uₙ(x)=uₙ(x₀)+∫_{x₀}^x u′ₙ(t)dt.FTC
3.By uniform convergence and Theorem 45, ∫_{x₀}^x u′ₙ(t)dt → ∫_{x₀}^x v(t)dt uniformly in x.swap limit and integral
4.Hence uₙ(x) converges uniformly to u(x)=A+∫_{x₀}^x v(t)dt.define limit function
5.Since v is a uniform limit of continuous functions u′ₙ, v is continuous.Theorem 44, under the usual continuity assumption on u′ₙ
6.By FTC, u is differentiable and u′(x)=v(x).differentiate integral
7.Therefore u′(x)=lim u′ₙ(x), and uₙ→u uniformly.conclusion
∎

Weierstrass M Test

Intuition Find a worst-case bound Mₖ for each term (no x). If the numbers ΣMₖ converge, every x is automatically controlled at the same rate — one N works for all x simultaneously.
Theorem 47 — Weierstrass M Test
If |uₖ(x)| ≤ Mₖ for all x and all k, and ΣMₖ converges, then Σuₖ(x) converges uniformly and absolutely. Why: |S(x)−Sₙ(x)| ≤ Σₖ₌ₙ₊₁^∞ Mₖ (tail of ΣMₖ → 0, no x dependence) Fails (silent) when ΣMₖ diverges — series may still converge, need Dirichlet.
Formal proof sketch
  • Control every function term by a number Mₖ independent of x.
  • The numerical tail ΣMₖ can be made smaller than ε.
  • Therefore the functional partial sums are uniformly Cauchy.
Strict formal proof
1.Let S_N(x)=Σ_{k=1}^N uₖ(x).partial sums
2.Because ΣMₖ converges, for every ε>0 there exists N such that q>p≥N ⇒ Σ_{k=p+1}^q Mₖ<ε.Cauchy criterion for numerical series
3.For every x and q>p≥N, |S_q(x)−S_p(x)| = |Σ_{k=p+1}^q uₖ(x)|.tail
4.By triangle inequality and |uₖ(x)|≤Mₖ, |S_q(x)−S_p(x)|≤Σ_{k=p+1}^q Mₖ<ε.uniform tail bound
5.Thus (S_N) is uniformly Cauchy.definition
6.The real-valued uniform Cauchy criterion gives uniform convergence of Σuₖ.completeness
7.Also Σ|uₖ(x)|≤ΣMₖ for every x, so the convergence is absolute for each x.absolute convergence
∎

Dirichlet Test for Uniform Convergence

Intuition When the natural bound gives ΣMₖ = Σ1/k = diverges (like Σsin(kx)/k), M test is silent. Dirichlet works instead: one factor aₖ(x) shrinks uniformly to 0, the other bₖ(x) has bounded partial sums. The oscillation and the shrinking envelope force uniform convergence.
Theorem 48 — Dirichlet Test for Uniform Convergence
If: aₖ(x) decreasing in k, aₖ(x) → 0 uniformly in x |Σₖ₌₁ⁿ bₖ(x)| ≤ M for all n and all x Then Σ aₖ(x)bₖ(x) converges uniformly. Key example: Σ sin(kx)/k on closed subsets of (0,2π): aₖ(x) = 1/k → 0 uniformly ✓ (no x dependence at all) |Σₖ₌₁ⁿ sin(kx)| ≤ 1/|sin(x/2)| ≤ M on any closed subset of (0,2π) ✓ → Σ sin(kx)/k converges uniformly on [δ, 2π−δ] for any δ > 0 Near x=0: sin(x/2)→0 so 1/|sin(x/2)|→∞ — condition 2 fails. Not uniform on all of (0,2π).
Formal proof sketch
  • Use the same Abel summation identity as the ordinary Dirichlet test.
  • The partial sums of bₖ(x) are bounded by one M for all x.
  • Uniform convergence aₖ(x)→0 makes the tail bound independent of x.
Strict formal proof
1.Let B_n(x)=Σ_{k=1}^n bₖ(x) and assume |B_n(x)|≤M for all n and all x.uniform bounded partial sums
2.For q>p, Abel summation gives Σ_{k=p+1}^q aₖ(x)bₖ(x)=a_q(x)B_q(x)−a_{p+1}(x)B_p(x)+Σ_{k=p+1}^{q-1}(aₖ(x)−a_{k+1}(x))B_k(x).summation by parts
3.Taking absolute values gives a bound ≤ M a_q(x)+M a_{p+1}(x)+MΣ_{k=p+1}^{q-1}(aₖ(x)−a_{k+1}(x)).triangle inequality
4.Since aₖ(x) decreases in k, the last sum telescopes to a_{p+1}(x)−a_q(x).monotone envelope
5.Therefore |Σ_{k=p+1}^q aₖ(x)bₖ(x)|≤2M a_{p+1}(x) for every x.tail estimate
6.Because aₖ→0 uniformly, choose p so large that sup_x a_{p+1}(x)<ε/(2M).uniform smallness
7.Then every tail is <ε for every x, so the series is uniformly Cauchy and hence uniformly convergent.uniform Cauchy criterion
∎
Theorem 49 — Power Series Uniform Convergence
Let Sₙ(x) = Σₖ₌₁ⁿ aₖ(x−c)ᵏ with radius of convergence R. |x−c| ≤ ρ < R ⟹ Sₙ(x) converges uniformly on |x−c| ≤ ρ Proof: |aₖ(x−c)ᵏ| ≤ |aₖ|ρᵏ = Mₖ, and ΣMₖ converges since ρ < R. Weierstrass M test applies. ⚠ Not uniform on the full open disk |x−c| < R — only on closed sub-disks.
Formal proof sketch
  • Inside a smaller closed interval |x−c|≤ρ<R, replace |x−c| by the fixed number ρ.
  • The resulting numerical majorant converges because ρ is still inside the radius.
  • Use the Weierstrass M Test.
Strict formal proof
1.Let |x−c|≤ρ<R.closed subinterval inside radius
2.For every k, |aₖ(x−c)^k|≤|aₖ|ρ^k.majorization
3.Since ρ<R, the numerical power series Σ|aₖ|ρ^k converges.definition of radius
4.Set Mₖ=|aₖ|ρ^k. Then |aₖ(x−c)^k|≤Mₖ and ΣMₖ converges.M-test setup
5.By the Weierstrass M Test, Σaₖ(x−c)^k converges uniformly and absolutely on |x−c|≤ρ.conclusion

Intuition Integration averages — small errors get smoothed out. Differentiation is local and sensitive — tiny wiggles can produce large derivative errors. So integration needs only uniform convergence of the functions; differentiation needs uniform convergence of the derivatives.

Formal proof sketch
  • Apply Theorem 45 to the sequence of partial sums.
  • The integral of a finite sum is the finite sum of integrals.
  • Then pass to the limit.
Strict formal proof
1.Let S_N(x)=Σ_{k=1}^N uₖ(x) and assume S_N→S uniformly on [a,b].series as partial sums
2.By Theorem 45, ∫_a^b S(x)dx = lim_{N→∞}∫_a^b S_N(x)dx.swap limit and integral
3.For each finite N, ∫_a^b S_N(x)dx = Σ_{k=1}^N ∫_a^b uₖ(x)dx.linearity for finite sums
