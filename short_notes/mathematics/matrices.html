Matrices

A matrix is not just a table of numbers — it is a machine that moves vectors. The columns record where the standard basis vectors land after the transformation. This perspective makes determinants, inverses, eigenvalues, and diagonalization geometrically clear.

The two key equations:

$A\mathbf{x}=\mathbf{b}$ (linear system)  ·  $A\mathbf{x}=\lambda\mathbf{x}$ (eigenvalue)

Row reduction solves the first; the characteristic polynomial solves the second.

What Is a Matrix?

A matrix is a rectangular array of numbers. Viewed as a linear transformation, it completely describes where every vector in the input space gets sent.

Definition — Matrix
An $m\times n$ matrix $A=(a_{ij})$ has $m$ rows and $n$ columns. It represents a linear transformation from $\mathbb{R}^n$ to $\mathbb{R}^m$: column $j$ records where basis vector $\mathbf{e}_j$ lands.
Transformation Intuition
To see what $A$ does to any vector, write it as a linear combination of basis vectors, then read off where those basis vectors went. That is exactly matrix multiplication.

Special Matrices

Certain matrix shapes have special properties that simplify computations enormously.

Square matrix
$m=n$. Determinant, inverse, and eigenvalues are only defined for square matrices.
Identity matrix $I_n$
Diagonal entries 1, off-diagonal 0. $AI=IA=A$ for any $A$. Acts like the number 1.
Zero matrix $O$
All entries 0. $A+O=A$, $AO=OA=O$.
Diagonal matrix
Non-zero entries only on main diagonal. Powers: just raise each diagonal entry to the power.
Upper/lower triangular
All entries below/above main diagonal are 0. Determinant = product of diagonal entries.
Symmetric matrix
$A=A^T$. All eigenvalues are real; eigenvectors for distinct eigenvalues are orthogonal.

Matrix Operations

Addition and scalar multiplication work entry-wise. Matrix multiplication is not commutative and encodes composition of transformations.

Definition — Matrix Multiplication
If $A$ is $m\times p$ and $B$ is $p\times n$, then $C=AB$ is $m\times n$ with $(C)_{ij}=\sum_{k=1}^p a_{ik}b_{kj}$ (row $i$ of $A$ dotted with column $j$ of $B$).
Properties of Multiplication
  1. $(AB)C=A(BC)$ — associative
  2. $A(B+C)=AB+AC$ — distributive
  3. $AB\ne BA$ in general — not commutative
  4. $(AB)^T=B^TA^T$ — transpose reverses order
Transformation Intuition
$AB$ means "first apply $B$, then apply $A$." The reversed order in $(AB)^T=B^TA^T$ mirrors the fact that to undo the composition, you undo $A$ first, then $B$.

Transpose, Symmetric & Skew-Symmetric

The transpose of $A$ flips it across the main diagonal. Symmetric and skew-symmetric matrices arise from this operation.

Definition — Transpose
$(A^T)_{ij}=A_{ji}$. Geometrically, if $A$ rotates and stretches space, $A^T$ is related to the reverse stretching.
Symmetric and Skew-Symmetric
  • Symmetric: $A=A^T$. Every real symmetric matrix is orthogonally diagonalizable.
  • Skew-symmetric: $A=-A^T$. Diagonal entries must all be 0.
  • Decomposition: Any square $A=\frac{A+A^T}{2}+\frac{A-A^T}{2}$ (symmetric + skew-symmetric).

Determinant

The determinant measures how much the transformation scales areas (in 2D) or volumes (in 3D). A zero determinant means the transformation collapses space — the matrix is singular.

Definition — Determinant ($2\times 2$)
$$\det\begin{pmatrix}a&b\\c&d\end{pmatrix}=ad-bc$$
Definition — Cofactor Expansion ($3\times 3$)
Expanding along row 1: $$\det(A)=a_{11}M_{11}-a_{12}M_{12}+a_{13}M_{13}$$ where $M_{ij}$ is the minor (determinant of the $2\times 2$ submatrix obtained by deleting row $i$, column $j$). Sign: $(-1)^{i+j}$.
Properties of Determinants
  1. $\det(AB)=\det(A)\det(B)$
  2. $\det(A^T)=\det(A)$
  3. Swapping two rows multiplies $\det$ by $-1$
  4. Multiplying a row by $k$ multiplies $\det$ by $k$, so $\det(kA)=k^n\det(A)$ for $n\times n$
  5. Adding a multiple of one row to another: $\det$ unchanged
  6. $\det(A^{-1})=1/\det(A)$
Consider the transformation interpretation: $AB$ scales area by $|\det(AB)|$. Applying $B$ first scales by $|\det B|$; then applying $A$ scales by $|\det A|$. The composition scales by the product, so $|\det(AB)|=|\det A||\det B|$. For the sign, one uses the fact that both sides are continuous and equal on all invertible matrices.
Geometric Meaning
$|\det A|$ = area of the image of the unit square (2D) or volume of the image of the unit cube (3D). $\det A > 0$ means orientation preserved; $\det A < 0$ means orientation reversed (like a reflection).

Adjoint & Inverse

The inverse undoes a transformation. It exists exactly when the determinant is non-zero — that is, when the transformation is not degenerate.

Definition — Cofactor and Adjoint
The cofactor $C_{ij}=(-1)^{i+j}M_{ij}$. The adjoint (classical adjugate) is $\text{adj}(A)=(C_{ij})^T$ — the transposed matrix of cofactors.
Theorem — Inverse Formula
If $\det(A)\ne 0$: $$A^{-1}=\frac{\text{adj}(A)}{\det(A)}$$ For a $2\times 2$ matrix: $A^{-1}=\dfrac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$.
Properties of the Inverse
  • $(AB)^{-1}=B^{-1}A^{-1}$ (reversal of order)
  • $(A^T)^{-1}=(A^{-1})^T$
  • $(A^{-1})^{-1}=A$

Systems of Linear Equations

Every system $A\mathbf{x}=\mathbf{b}$ can be analysed by forming the augmented matrix $[A|\mathbf{b}]$ and row-reducing.

Theorem — Cramer's Rule (small systems)
For $A\mathbf{x}=\mathbf{b}$ with $n\times n$ invertible $A$: $x_i=\dfrac{\det(A_i)}{\det(A)}$, where $A_i$ replaces column $i$ of $A$ with $\mathbf{b}$.
Solution Existence and Uniqueness
  • Unique solution: $\text{rank}(A)=\text{rank}([A|\mathbf{b}])=n$
  • Infinitely many solutions: $\text{rank}(A)=\text{rank}([A|\mathbf{b}])
  • No solution: $\text{rank}(A)<\text{rank}([A|\mathbf{b}])$
Transformation Intuition
$A\mathbf{x}=\mathbf{b}$ asks: "what input vector gets mapped to $\mathbf{b}$?" If $A$ collapses dimensions ($\det A=0$), the output space is smaller than the input — some $\mathbf{b}$s are unreachable, some have many pre-images.

Echelon Form & Rank

Row reduction reduces a matrix to echelon form. The rank — the number of non-zero rows after reduction — captures the "effective dimension" of the transformation.

Definition — Row Echelon Form (REF)
A matrix is in REF if: (i) all zero rows are at the bottom; (ii) each leading non-zero entry (pivot) in a row is to the right of the pivot above it.
Definition — Reduced REF (RREF)
REF plus: (i) each pivot is 1; (ii) every entry above a pivot is 0.
Definition — Rank
$\text{rank}(A)$ = number of non-zero rows in the echelon form = dimension of the column space. For an $m\times n$ matrix: $\text{rank}(A)\le\min(m,n)$.
Rank–Nullity Theorem
For $A$ of size $m\times n$: $\text{rank}(A)+\text{nullity}(A)=n$. Nullity = number of free variables = dimension of the null space (set of all $\mathbf{x}$ with $A\mathbf{x}=\mathbf{0}$).

Eigenvalues & Eigenvectors

Eigenvectors are the "special directions" that a matrix only scales, never rotates. Finding them reveals the essential structure of the transformation.

Definition — Eigenvalue / Eigenvector
$\lambda$ is an eigenvalue of $A$ and $\mathbf{x}\ne\mathbf{0}$ is a corresponding eigenvector if $A\mathbf{x}=\lambda\mathbf{x}$.
Theorem — Characteristic Equation
The eigenvalues are the roots of the characteristic polynomial: $$\det(A-\lambda I)=0$$ For each $\lambda$, the eigenvectors are found by solving $(A-\lambda I)\mathbf{x}=\mathbf{0}$.
$A\mathbf{x}=\lambda\mathbf{x}$ rearranges to $(A-\lambda I)\mathbf{x}=\mathbf{0}$. For a non-trivial solution, the matrix $A-\lambda I$ must be singular, i.e. $\det(A-\lambda I)=0$.
Properties
  • $\sum\lambda_i = \text{tr}(A)$ (sum of eigenvalues equals trace)
  • $\prod\lambda_i = \det(A)$ (product of eigenvalues equals determinant)
  • Symmetric matrices: all eigenvalues are real; eigenvectors for different eigenvalues are orthogonal
  • $A$ is invertible $\iff$ 0 is not an eigenvalue

Orthogonal Matrices

Orthogonal matrices represent rigid rotations and reflections. They preserve distances and angles — and they are the easiest matrices to invert.

Definition — Orthogonal Matrix
A square matrix $Q$ is orthogonal if $Q^TQ=QQ^T=I$, equivalently $Q^{-1}=Q^T$. Its columns form an orthonormal set.
Key Properties
  • $|Q\mathbf{x}|=|\mathbf{x}|$ — lengths preserved
  • $Q\mathbf{x}\cdot Q\mathbf{y}=\mathbf{x}\cdot\mathbf{y}$ — angles preserved
  • $\det(Q)=\pm 1$ — $+1$ for rotation, $-1$ for reflection

Gram–Schmidt Process

Given linearly independent vectors $\mathbf{v}_1,\mathbf{v}_2,\ldots$, produce orthonormal vectors by subtracting projections:

$$\mathbf{u}_k=\mathbf{v}_k-\sum_{j=1}^{k-1}\text{proj}_{\mathbf{u}_j}\mathbf{v}_k, \qquad \mathbf{e}_k=\frac{\mathbf{u}_k}{|\mathbf{u}_k|}$$

Similarity & Diagonalization

Diagonalization rewrites a matrix in the eigenvector basis, where the transformation reduces to simple scalings. This makes powers and functions of matrices easy.

Definition — Similar Matrices
$A$ and $B$ are similar if $B=P^{-1}AP$ for some invertible $P$. Similar matrices represent the same transformation in different bases.
Theorem — Diagonalization
An $n\times n$ matrix $A$ is diagonalizable iff it has $n$ linearly independent eigenvectors. Then: $$A=PDP^{-1}$$ where the columns of $P$ are eigenvectors and $D$ is diagonal with eigenvalues on the diagonal.
If $\mathbf{p}_1,\ldots,\mathbf{p}_n$ are independent eigenvectors with eigenvalues $\lambda_1,\ldots,\lambda_n$, form $P=[\mathbf{p}_1|\cdots|\mathbf{p}_n]$. Then $AP=[A\mathbf{p}_1|\cdots|A\mathbf{p}_n]=[\lambda_1\mathbf{p}_1|\cdots|\lambda_n\mathbf{p}_n]=PD$. Since $P$ is invertible (independent columns), multiply right by $P^{-1}$: $A=PDP^{-1}$.
Powers of a Diagonalizable Matrix
$A^n=PD^nP^{-1}$. Since $D$ is diagonal, $D^n$ just raises each eigenvalue to the $n$th power. For symmetric matrices: $A^n=PD^nP^T$.
Intuition
$P^{-1}$ changes coordinates to the eigenvector basis — the natural frame for this transformation. In that basis, $D$ just scales each axis independently. Then $P$ maps back. Powers become trivial: scaling by $\lambda$ repeatedly gives $\lambda^n$.
Worked example — powers using diagonalisation

Find a formula for $A^n$

Let $A=\begin{pmatrix}2&1\\1&2\end{pmatrix}$. Eigenvectors $\mathbf{v}_1=(1,1)^T$, $\mathbf{v}_2=(1,-1)^T$ with eigenvalues $3$ and $1$. Hence:

$$P=\begin{pmatrix}1&1\\1&-1\end{pmatrix},\quad D=\begin{pmatrix}3&0\\0&1\end{pmatrix},\quad A=PDP^{-1}$$
Show the calculation

Since $P^{-1}=\tfrac12\begin{pmatrix}1&1\\1&-1\end{pmatrix}$,

$$A^n=PD^nP^{-1}=\begin{pmatrix}1&1\\1&-1\end{pmatrix}\begin{pmatrix}3^n&0\\0&1\end{pmatrix}\frac{1}{2}\begin{pmatrix}1&1\\1&-1\end{pmatrix}$$
$$\boxed{A^n=\frac{1}{2}\begin{pmatrix}3^n+1&3^n-1\\3^n-1&3^n+1\end{pmatrix}}$$

Repeated multiplication disappears because the diagonal matrix only raises eigenvalues to powers.

Diagonalisation facts to remember
  • If $A=PDP^{-1}$, then $A^n=PD^nP^{-1}$. For symmetric matrices with orthonormal eigenvectors, $A=PDP^T$, so $A^n=PD^nP^T$.
  • The columns of $P$ must be eigenvectors in the same order as the eigenvalues in $D$.
  • $\det(A)\ne 0$ only means $A$ is invertible — it does not automatically mean $A$ is diagonalizable.
  • A zero eigenvalue makes the matrix singular, but it can still be diagonalizable if there are enough independent eigenvectors.
  • A real matrix is orthogonally diagonalizable exactly when $A=A^T$.
Extra exam points
  • RREF works for non-square matrices. For a $4\times 3$ matrix, maximum rank is 3.
  • Column operations preserve rank but do not generally preserve the column space.
  • Swapping two rows changes the sign of the determinant.
  • $\det(A^T)=\det(A)$.
  • For a $3\times 3$ matrix, $\det(2A)=2^3\det(A)=8\det(A)$.
